# CS170-Project-1
A program that solves the eight-puzzle in three ways. Uniform Cost Search, A* with the Misplaced Tile heuristic, and A* with the Manhattan Distance heuristic.

# Installation & Setup

## 1. Install Python

This project was written using **Python 3.10 or newer**.

To check if Python is installed:

```bash
python3 --version
```

If Python is not installed, download it from:

https://www.python.org/downloads/

---

## 2. Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

(Replace with your actual repository URL.)

---

## 3. Install Required Libraries

This project requires the following Python libraries:

- pandas  
- seaborn  
- matplotlib  

Install them using:

```bash
pip install pandas seaborn matplotlib
```

If `pip` does not work, try:

```bash
pip3 install pandas seaborn matplotlib
```

---

## 4. Run the Program

From inside the project directory, run:

```bash
python main.py
```

or

```bash
python3 main.py
```
